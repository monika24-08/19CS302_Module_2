# EX 6 C Program to print the string "KEYBOARD" n number of times.
## DATE:
## AIM:
To write a C Program to print the string "KEYBOARD" n number of times.

## Algorithm
1.Start the program.
2.Declare an integer variable n.
3.Read the value of n from the user.
4.Use a loop to print "KEYBOARD" n times.
5.End the program.

## Program:
```
/*
Program to print the string "KEYBOARD" n number of times.
Developed by: MONIKA M
RegisterNumber: 212223060169
*/

#include <stdio.h>

int main() {
    int n, i;

    printf("Enter how many times to print: ");
    scanf("%d", &n);

    for(i = 0; i < n; i++)
    {
        printf("KEYBOARD\n");
    }

    return 0;
}

```
